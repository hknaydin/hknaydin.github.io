AÇIKLAMA
Hanoi Kulesi: N diski A'dan C'ye taşı. B yardımcı kule. Büyük disk küçüğün üstüne konamaz.
def hanoi(n, src, aux, dst):
# Base case: tek disk
if n == 1:
move(src → dst) # ♟ Taşı
return
 
# Adım 1: Üst n-1 diski aux'a taşı
hanoi(n-1, src, dst, aux)
 
# Adım 2: En büyük diski dst'ye taşı
move(src → dst) # ♟ En büyük
 
# Adım 3: n-1 diski dst'ye taşı
hanoi(n-1, aux, src, dst)

PROBLEM

N diski A kulesinden C kulesine taşı. Kural: Büyük disk küçüğün üstüne konamaz. B kule yardımcı olarak kullanılabilir.

ÖZYINELEME MANTIĞI

1) Üst n−1 diski yardımcıya taşı
2) En büyük diski hedefe taşı
3) n−1 diski yardımcıdan hedefe taşı

T(n) = 2·T(n−1) + 1
