N-Queens Problemi: N×N satranç tahtasına N adet vezir yerleştir; hiçbiri birbirini tehdit etmesin.

Backtracking: Her satıra sırayla bir vezir koy. Geçersizse geri dön ve farklı sütun dene.
def solve(row):
# Tüm vezirler yerleşti → çözüm!
if row == N:
return True # ✓ ÇÖZÜM BULUNDU
 
for col in range(N):
# Bu pozisyon geçerli mi?
if is_safe(row, col):
board[row] = col # Vezir koy ♛
if solve(row + 1): # Özyineleme
return True
board[row] = None # ← GERİ AL
 
return False # Bu yol başarısız
def is_safe(row, col):
for r in range(row):
# Aynı sütun kontrolü
if board[r] == col: return False
# Çapraz kontrol
if abs(board[r]-col) == abs(r-row):
return False
return True # Güvenli!

PROBLEM NEDİR?

N×N satranç tahtasına N vezir yerleştir. Hiçbir vezir diğerini tehdit etmesin. Vezir; yatay, dikey ve çapraz yönlerde saldırır.

BACKTRACKING NEDİR?

Geri izleme algoritması, çözümü adım adım inşa eder. Bir adım çıkmaza girince geri döner ve farklı bir yol dener. Bu sayede tüm olasılıkları sistematik tarar.

✓ Yerleştir → İlerle
✗ Çıkmaz → Geri Al

ZAMAN KARMAŞIKLİĞİ

En kötü durum: O(N!). Ancak kırpma (pruning) sayesinde çok daha hızlı çalışır.
