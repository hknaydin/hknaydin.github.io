ASK
Genlik Kaydırmalı Anahtarlama
Amplitude Shift Keying
Formül
s(t) = A(t)·cos(2πfct)

A = A₁ → bit = 1
A = A₀ → bit = 0
Çalışma Prensibi
Taşıyıcı sinyal sabit frekansta salınır
Bit '1' için yüksek genlik, '0' için düşük (veya sıfır) genlik kullanılır
OOK (On-Off Keying) en basit ASK türüdür
Teknik Özellikler

FSK
Frekans Kaydırmalı Anahtarlama
Frequency Shift Keying
Formül
s(t) = A·cos(2πf(t)·t)

f = f₁ → bit = 1
f = f₀ → bit = 0

Δf = f₁ − f₀
Çalışma Prensibi
Genlik sabit tutulur; frekans veriyle değişir
Bit '1' için yüksek frekans, '0' için düşük frekans
Gürültüye karşı ASK'tan çok daha dayanıklıdır
Teknik Özellikler
Bant genişliğiB = 2(Δf + fm)
Gürültü bağışıklığıOrta
UygulamaRadyo, modem
Verimlilik1 bit/Hz

PSK
Faz Kaydırmalı Anahtarlama
Phase Shift Keying
Formül (BPSK)
s(t) = A·cos(2πfct + φ(t))

φ = 0° → bit = 1
φ = 180° → bit = 0
PSK Çeşitleri
BPSK: 2 faz (0°, 180°) — 1 bit/sembol
QPSK: 4 faz (0°, 90°, 180°, 270°) — 2 bit/sembol
8-PSK: 8 faz — 3 bit/sembol
Teknik Özellikler
Gürültü bağışıklığıYüksek
UygulamaWiFi, LTE
BPSK verimi1 bit/Hz
QPSK verimi2 bit/Hz

QAM
Kuadratür Genlik Modülasyonu
Quadrature Amplitude Modulation
Formül (16-QAM)
s(t) = I·cos(2πfct)
    − Q·sin(2πfct)

I: In-phase bileşen
Q: Quadrature bileşen
QAM Seviyeleri
4-QAM = QPSK (2 bit/sembol)
16-QAM: 16 nokta, 4 bit/sembol
256-QAM: 8 bit/sembol (WiFi 6)
1024-QAM: 10 bit/sembol (5G NR)
