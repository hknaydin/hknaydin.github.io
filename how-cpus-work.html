OUT-OF-ORDER EXEC
CPU reorders instructions to avoid stalls. Tomasulo Algorithm tracks dependencies via reservation stations.
SUPERSCALAR
Multiple execution units run in parallel. Modern CPUs fetch/decode 4–6 instructions per cycle.
SPECULATIVE EXEC
CPU executes instructions before knowing if they're needed. Results discarded if prediction wrong (Spectre/Meltdown).
SMT / HYPERTHREADING
Two logical cores share one physical core. Fills pipeline stall slots with another thread's instructions.
