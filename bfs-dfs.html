BFS — GENİŞLİK ÖNCE ARAMA
Veri Yapısı: Kuyruk (Queue)
FIFO — ilk eklenen ilk işlenir
Her zaman en kısa yolu bulur
Ağırlıksız graf için optimal
O(V+E) bellek kullanır
Büyük grafta yavaş olabilir
Dalga gibi yayılır
Eşit mesafeleri önce ziyaret eder
DFS — DERİNLİK ÖNCE ARAMA
Veri Yapısı: Yığın (Stack)
LIFO — son eklenen ilk işlenir
En kısa yolu garanti etmez
İlk bulduğu yolu döndürür
Az bellek kullanır
Yalnızca aktif yolu saklar
Derine dalar
Bir yolu sonuna kadar keşfeder
BFS — KUYRUK (QUEUE)
def bfs(grid, start, end):
queue = deque([start]) # Kuyruk
visited = {start} # Ziyaret edildi
parent = {start: None}
 
while queue: # Kuyruk doluysa
node = queue.popleft() # ← Sol'dan al
if node == end: break # Hedefe ulaştı!
for nb in neighbors(node):
if nb not in visited:
visited.add(nb) # Ziyaret et
parent[nb] = node
queue.append(nb) # Sağa ekle →
return reconstruct(parent)
DFS — YIĞIN (STACK)
def dfs(grid, start, end):
stack = [start] # Yığın (Stack)
visited = {start} # Ziyaret edildi
parent = {start: None}
 
while stack: # Yığın doluysa
node = stack.pop() # ← Üstten al
if node == end: break# Hedefe ulaştı!
for nb in neighbors(node):
if nb not in visited:
visited.add(nb) # Ziyaret et
parent[nb] = node
stack.append(nb) # Üste ekle ↑
return reconstruct(parent)
